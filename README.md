## Installation:
  ##### Clone the repository
  ```bash
  git clone https://github.com/SparshJain2000/Pokedex.git
  ```
  ##### Install typescript globally
  ```bash
  npm i -g typescript 
  ```
  ##### compile typescript file
  ```bash
  tsc -w
  ```
  ##### or
  ```bash
  tsc app.ts
  ```
  ##### Open index.html in any browser
